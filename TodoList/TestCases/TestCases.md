## TC-14: Корректное отображение при разных флагах.

**Описание:**
проверка работы команды view с флагами -i, -s, -d, -a, --index, --status, --update-date, --all

**Предусловие:**
Программа запущена, авторизация выполнена, добавлены некоторые задачи.

**Последовательность действий:**
1. Для проверки флага индекса, ввести команду view -i, либо view --index.
2. Для проверки флага статутов, ввести команду view -s, либо view --statuses.
3. Для проверки флага времени и даты, ввести команду view -d, либо view --update-date
4. Для того, чтобы показать все флаги сразу, ввести команду view -a, либо view --all.

**Ожидаемый результат**:
Программа будет корректно показывать каждый из выбранных флагов. При выборе индекса будет показаны номера задач. На втором пункте будет показано на каком этапе сейчас задача (не начато, в процессе, выполнено, отложено, провалено). При выборе флага времени и даты, будует показана дата создания задачи. Если установить флаг, показывающий все сразу, программа будет выводить индекс, статус и дату одновременно.
